# Route Seeking

## Prerequisite Concepts

- Arrays or lists
- Basic loops
- Simple graph vocabulary
- Reading small datasets

## Target Concepts

- Graph nodes and edges
- Searching neighbors
- Tracking visited nodes
- Reconstructing a path

## What To Inspect

Inspect a small map represented as connected places. Notice how each place
stores its neighbors before adding a search algorithm.

## What To Modify Or Build

Build a small route search that finds a path between two places. Start with a
tiny graph, then add one more connection and verify that the result changes.
