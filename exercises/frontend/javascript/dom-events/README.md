# DOM Events

## Prerequisite Concepts

- HTML buttons and inputs
- CSS classes
- JavaScript variables
- DOM selection

## Target Concepts

- Event listeners
- Reading input values
- Updating text content
- Toggling classes

## What To Inspect

Inspect a simple page with a button, an input, and a message area. Identify
which elements JavaScript needs to select.

## What To Modify Or Build

Build a small interaction where a button reads text from an input and updates a
message on the page. Add a second button that clears the message.
